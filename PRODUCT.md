# PRODUCT

## 产品定位

薪跳 PayDance 是一款面向打工人的桌面实时薪资仪表盘。它把“今天已经挣了多少钱”变成一个安静常驻、实时跳动、可以放在屏幕角落扫一眼的桌面看板。

## 核心用户

- 使用 Windows 桌面办公、希望更直观看见时间与收入关系的用户。
- 喜欢轻量、克制、无需账号、不上传数据的桌面小工具用户。
- 需要可置顶、托盘常驻、迷你悬浮金额窗口的办公场景用户。

## 核心体验

- 主窗口优先展示今日入账金额，其次展示已工作、距离下班和今日预计。
- 迷你悬浮窗口只展示金额，不承载次级信息；它是稳定体验，不随主界面实验频繁改变。
- 设置中心用于配置薪资、作息、显示偏好和作者归属，不做营销页。
- 所有薪资与作息数据只保存在本机，不做登录、云同步、遥测或广告。

## 产品边界

- 不开发快捷键功能。
- 不开发提醒功能。
- 不开发分段时间轴。
- 不把迷你悬浮模式改成复杂面板。
- 不在主界面堆叠超过必要层级的信息面。

## 平台策略

当前正式验证平台是 Windows 11。Vue 3、TypeScript、薪资计算逻辑和大部分 UI 状态可复用到 macOS、Linux、移动端或小程序重写，但托盘、置顶、透明窗口、迷你悬浮和系统材质需要按平台重新设计。
