# v0.5 Productization Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Upgrade the desktop salary ticker into v0.5 with salary input modes, weekly workday rules, onboarding, clearer status text, corrected mini-mode topmost behavior, and versioned settings migration.

**Architecture:** Keep salary calculation as pure TypeScript functions with focused tests. Store settings through the existing Tauri Store composable, migrate old config into a v3 shape, and keep UI work in Vue components that consume derived state from `App.vue`.

**Tech Stack:** Vue 3, TypeScript, Vitest, Tauri 2, Rust, Tauri Store.
